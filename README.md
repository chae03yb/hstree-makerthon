## 실행 방법

### 1. 환경 설정
1. **프로젝트 클론**
   ```bash
   git clone https://github.com/chae03yb/hstree-makerthon.git mandarin-in-snack
   cd mandarin-in-snack
   ```

2. **필요한 패키지 설치**
   ```bash
   python3 setup.py
   ```

### 2. 실행
1. **개발 서버 실행**
   ```bash
   cd webserver
   python main.py
   ```

### 3. 접속
   - 로컬에서: `http://localhost:8720`
